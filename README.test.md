# confirm.attr
Enhance the behaviour of buttons and forms

## Ussage
```html
<form u1-confirm="Save name?" target="_blank">
        <label>Name: <input name="x"></label>
        <button>save</button>
    </form>
```

## Install
```html
<script src="https://cdn.jsdelivr.net/gh/u1ui/confirm.attr@1.0.0/confirm.min.js" type=module>
```
